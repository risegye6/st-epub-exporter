# 📚 SillyTavern EPUB Exporter

실리태번의 채팅 기록을 깔끔한 전자책(EPUB) 파일로 추출해 주는 확장기능입니다. 
추출한 파일은 모바일 기기나 PC의 이북 리더기(리디, 캘리버)에서 소설책처럼 편하게 읽을 수 있습니다!

## ✨ 주요 기능
* 마크다운 지원: 굵은 글씨, 기울임꼴, 코드블럭 등 실리태번의 텍스트 감성을 그대로 유지합니다.
* 아바타 프사 : 말풍선 옆에 유저와 캐릭터의 프로필 이미지를 함께 띄워줍니다.
* 에셋 출력 : 에셋 폴더를 자동으로 인식하여 파일 생성시 함께 저장됩니다.  
* 챕터 분할 기능: 원하는 채팅 구간만 잘라서 챕터별로 나눌 수 있습니다.
* 시스템 텍스트 제거: 상태창이나 선택지 같은 시스템 메시지는 몰입을 방해하지 않게 제거하고 본문만 추출합니다.
* 다양한 봇 환경 지원: 단일 봇뿐만 아니라 다인 봇, 시뮬레이터 봇에서 모두 생성 가능합니다. 
---

## 🚀 사용 방법
1. 실리태번 메뉴의 **마법봉(Extensions) 아이콘**을 클릭합니다.
2. 목록에서 **[ EPUB 내보내기 ]** 버튼을 클릭하여 설정 창을 엽니다.
3. 옵션을 원하는 대로 설정한 뒤 **[ 다운로드 ]** 버튼을 누르면 `.epub` 파일이 생성됩니다!

### ⚙️ 상세 설정 가이드
* 1. 챕터 지정 (선택)
  전체를 한 번에 뽑지 않고, 구간을 나누고 싶을 때 사용합니다.
  아래 형식에 맞춰 작성해 주세요. (비워두면 처음부터 끝까지 한 번에 추출됩니다)
  > 작성 예시:
  > 프롤로그 : 0-50
  > 1장 만남 : 51-120
  > 2장 위기 : 121-200

* 2. 에셋 이미지 경로 지정
캐릭터가 채팅 중에 띄워주는 인라인 이미지를 이북 파일 안에 함께 압축해 넣기 위한 경로입니다.
기본적으로 /characters/캐릭터이름/으로 자동 세팅되어 있으며, 만약 에셋 이미지가 다른 전용 폴더에 들어있다면 해당 경로로 수정해 주셔야 엑스박스 없이 이미지가 정상적으로 출력됩니다.

* 3. 마크다운 디자인
  마크다운에 쓰일 글자 색상(이탤릭 및 볼드)을 직접 지정할 수 있습니다. 기본적으로 깔끔한 다크 브라운/올리브 그린 계열로 세팅되어 있습니다.

* 4. 아바타 프사 포함
  체크 시 채팅 로그에 캐릭터와 유저의 동그란 프로필 이미지가 추가됩니다. 

---

## ⚠️ 주의사항
* 눈가림 채팅 주의: 채팅 로그에서 눈이 가려진(hidden) 메시지는 반드시 **unhide** 처리로 풀어주어야 정상적으로 추출됩니다.
* 내보내기 용량: 채팅 로그가 수천~수만 줄 단위로 엄청나게 길거나 포함된 이미지가 많을 경우, 압축 변환 과정에서 수 초 정도 딜레이가 발생할 수 있습니다. 버튼이 "변환 중..."으로 바뀌었다면 잠시만 기다려 주세요!
* 폰트 관련: 인터넷에서 실시간으로 폰트를 다운로드하는 방식이 아닌, **'기기에 설치된 폰트'**를 불러오는 방식입니다. PC나 모바일에 미리 [Pretendard] 폰트를 설치해 두시면 훨씬 더 예쁜 결과물을 보실 수 있습니다.
* 뷰어 호환성: 캘리버(Calibre), 리디(RIDI) 에서 최적의 레이아웃으로 테스트되었습니다. 이외의 이북앱에서는 레이아웃이 깨질 수 있습니다. 리더기 자체의 '사용자 CSS 강제 적용' 옵션이 켜져 있다면, 이 확장기능의 디자인이 일부 무시될 수 있으니 뷰어 설정을 확인해 주세요.
